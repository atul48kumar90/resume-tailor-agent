## Color Palette

- **Primary Blue**: #2563EB (Buttons, links, highlights)
- **Success Green**: #10B981 (Success states, positive metrics)
- **Warning Orange**: #F59E0B (Warnings, medium scores)
- **Error Red**: #EF4444 (Errors, low scores, missing items)
- **Background**: #F9FAFB (Light gray background)
- **Card White**: #FFFFFF (Card backgrounds)
- **Text Primary**: #111827 (Dark gray for headings)
- **Text Secondary**: #6B7280 (Medium gray for body text)

## Typography

- **Headings**: Inter, Bold, 24-48px
- **Body**: Inter, Regular, 14-16px
- **Small Text**: Inter, Regular, 12-14px

## Component Specifications

### Buttons
- Primary: Blue background, white text, rounded-lg, px-6 py-3
- Secondary: Gray border, gray text, rounded-lg, px-6 py-3
- Icon buttons: Circular, 40px, with icon centered

### Cards
- White background, rounded-lg, shadow-md, padding: 24px
- Hover effect: Slight shadow increase

### Input Fields
- Border: 1px solid #D1D5DB
- Focus: 2px blue ring, border becomes blue
- Rounded: rounded-md
- Padding: px-4 py-3

### File Upload Areas
- Dashed border: 2px, gray-300
- Hover: border-blue-500, bg-blue-50
- Active: border-blue-600, bg-blue-100
